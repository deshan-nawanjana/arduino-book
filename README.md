# Arduino Programming Tutorial in Sinhala

Arduino Book is a web-based tutorial project designed to teach Arduino programming in the Sinhala language. It provides a structured learning path covering fundamentals, coding concepts, and practical electronics projects, making it easier for beginners and students to understand embedded systems in their native language. The platform includes step-by-step guides, examples, and hands-on experiments to bridge theory and real-world application.

Launch Now: [https://deshan-nawanjana.github.io/arduino-book/](https://deshan-nawanjana.github.io/arduino-book/)

## Key Features

- Read about the introduction and fundamentals of Arduino.
- Simply explained A-Z tutorial segments.
- Each segment is included with an example code, and board and module setup.

## Libraries and Frameworks Used

- [Vue.js](https://vuejs.org/) - Progressive framework for building user interfaces
