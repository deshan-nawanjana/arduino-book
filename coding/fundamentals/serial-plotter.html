Serial Plotter මගින් දත්ත නිරූපණය

By Deshan Nawanjana

Serial Monitor මගින් තිරය මත දත්ත නිරූපණය කිරීම අක්ෂර සහ අංක මගින් සිදු වේ. නමුත් ඔබට ප්‍රස්ථාරික ආකාරයෙන් දත්ත නිරූපණයට අවශ්‍ය නම් ඒ සඳහා Serial Plotter භාවිතා කිරීමට සිදු වේ. Tools > Serial Plotter මගින් මෙම මෙවළම විවෘත කළ හැකි අතර Serial Monitor මෙමගින් ඔබ දර්ශණය කිරීමට බලාපොරොත්තු වන දත්තයෙහි සංඛ්‍යාත්මක අගය ගෙන එය කාලයට අනුරූපව විචලනය වන අයුරින් ප්‍රස්ථාරයක දැක්වීම සිදු වේ. මෙහි එන නිදසුනේ දැක්වෙනුයේ i නමැති විචල්‍යයක් 0 සිට 50 දක්වාත් නැවත 0 දක්වාත් විචලනය වීම පරිගණකයේ දර්ශණය වීමට සැලැස්වීමයි. එය Serial Moitor මගින් සාමාන්‍ය ආකාරයෙන් එකින් එක පහළින් පහළට සංඛ්‍යාත්මකව දර්ශණය කරන අතරම Serial Plotter මගින් එය කාලයට අනුරූපව ප්‍රස්ථාරික ආකරයෙන් දර්ශණය කරයි. මෙහිදී ඔබට Serial Monitor සහ Serial Plotter එක වර විවෘත කළ නොහැකි අතර එක් වරකට භාවිත කළ හැක්කේ ඉන් එකක් පමණි.

REQUIREMENTS
1 x ARDUINO BOARD
SETUP
Connect UNO Board to your PC 
void setup() {
  Serial.begin(9600);
}

void loop() {
  for (int i = 0; i < 50; i++) {
    Serial.println(i);
  }
  for (int i = 49; i > -1; i--) {
    Serial.println(i);
  }
}
