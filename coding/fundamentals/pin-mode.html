PIN MODE හඳුන්වාදීම

By Deshan Nawanjana

ඔබ දන්නා ආකාරයට Ardiuno පුවරුව මත වැඩි භාවිතයක් ඇති Pins 13ක් පවතී මෙම Pins සමග විවිධ ඉලෙක්ට්‍රොනික උපාංග සම්බන්ධ කරනු ලැබේ. එසේ සම්බන්ධ කිරීමේදී එම Pins යොදාගන්නේ ආදානය සඳහාද, නැතිනම් ප්‍රතිදානය සඳහාද යන්න ප්‍රථමයෙන් දැක්විය යුතුය. නිදසුනක් වශයෙන් 6වන Pin එකට LEDය + අග්‍රය සම්බන්ධ වේ නම් එම Pin එක ප්‍රතිදාන Pin එකක් ලෙස දැක්විය යුතුය. (LED මගින් විදුලිය වැය කර ආලෝකය නිකුත් කරන නිසා එය පුවරුවෙන් දත්ත ප්‍රතිදානය කරන සංරචකයක් ලෙස හැඳින්වේ) ඔබ 3වන Pin එකට Ultrasonic Sensor එකෙහි Echo අග්‍රයක් සම්බන්ධ කරයි නම් එම Pin එම පුවරුවට දත්ත ආදානය කරන Pin එකක් ලෙස හඳුන්වා දිය යුතුය. (Ultrasonic Sensor එකෙහි Echo අග්‍රය මගින් ඉදිරියෙන් ඇති බාධකයකට ඇති දුර මැනීමට අවශ්‍ය දත්ත ලබා දේ)

මේ ආකාරයට යම් කිසි Pin එකක් ආදානය හෝ ප්‍රතිදානය සඳහා යොදාගන්න විට කලින් එය හඳුන්වා දීමට pinMode(PIN_ID,TYPE) යන කේතය භාවිත කරයි. මෙහි PIN_ID ලෙස Pin එකෙහි අංකයත් TYPE සඳහා INPUT හෝ OUTPUT යන කේත වචන වලින් එකක් යොදා ගැනේ. බොහෝ විට මෙම හැඳින්වීම කරනුයේ setup කොටස තුළයි. නමුත් ඔබට අවශ්‍ය විට එය loop කොටස තුළද ඇතුලත් කළ හැකිය.

1 x ARDUINO BOARD
