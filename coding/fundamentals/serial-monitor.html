Serial Monitor සමග දත්ත හුවමාරුව

By Deshan Nawanjana

Arduino පුවරුව පරිගණකයට සම්බන්ධ කර ඇති අවස්ථාවක පුවරුව හා පරිගණකය අතර දත්ත සම්ප්‍රේෂණය කිරීමේ හැකියාවක් පවතී. Arduino IDE මෘදුකාංගයේ ඇති මෙම මෙවලම Serial Monitor නම් වේ. එය Arduino IDE තිරයේ දකුණු ඉහළ කෙලවරේ ඇති බටනය මගින් විවෘත කළ හැකිය.

මේ සඳහා මුලින්ම setup යන කොටසේ Serial.begin(BAUD_RATE) ලෙස කේතයක් යෙදීමෙන් ස්ම්ප්‍රේෂණ මාරගය විවෘත කළ යුතුය. මෙහිදී බහුලව යොදාගන්න BAUD_RATE එක වන්නේ 9600 යි. පසුව ඔබට අවශ්‍ය අවස්ථාවකදී පුවරුවෙන් දත්ත යැවීමට Serial.println(DATA) භාවිත කළ හැකිය. මේ ආකාරයෙන් පරිගණකය වෙත යවනු ලබන දත්ත Serial Monitor එකෙහි පිළිවෙලින් මුද්‍රණය වේ. ඔබට අවශ්‍ය ඕනෑම ආකාරයක දත්තයක් (ප්‍රතිසම හෝ අංකිත කියවුමක්, පණිවුඩයක්) Serial Monitor වෙත සම්ප්‍රේෂණය කළ හැකිය.එමෙන්ම Serial Monitorහි ඉහලින් ඇති Textbox එක මත ඔබට අවශ්‍ය දත්තයක් හෝ පණිවුඩයක් යතුරු ලියනය කර Send Button එක මගින් පුවරුව වෙත සම්ප්‍රේෂණය කළ හැකිය. එම දත්ත පුවරුව මගින් කියවීම කරනුයේ Serial.read() විධානය මගිනි. කෙසේ වෙතත් එම කෙතයට පෙර පණිවුඩ ඇති දැයි Serial.available() මගින් හඳුනාගෙන සිටිය යුතුය.

REQUIREMENTS
1 x ARDUINO BOARD
SETUP
Connect UNO Board to your PC 
void setup() {
  Serial.begin(9600);
}

void loop() {
  int snd_data = analogRead(6);
  Serial.println(snd_data);

  String rec_data;
  if (Serial.available() > 0) {
    rec_data = Serial.read();
  }
}
