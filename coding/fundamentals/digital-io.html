DIGITAL I/O (අංකිත ආදාන/ප්‍රතිදාන)

By Deshan Nawanjana

අංකිත (Digital) ආදානයක් හෝ ප්‍රතිදානයක් සාමාන්‍ය පරිදි 1 හෝ 0 ආකාරයේ වේ. එනම් විදුලිය තිබීම හෝ නොතිබීමයි. UNO පුවරුවේ ඇති Pins 13 හි විභවය අඩු හෝ වැඩි කිරීමට මෙහි එන කේත කොටස් භාවිතා වේ. නිදසුනක් ලෙස LEDයක් දැල්වීමට අවශ්‍ය නම් එම LEDයේ + අග්‍රය සම්බන්ධ Pin එකෙහි විභවය වැඩි කළ යුතුය. මේ සඳහා digitalWrite(PIN_ID,STATUS) යන කේතය යොදා ගැනේ. PIN_ID සඳහා ඔබය අවශ්‍ය Pin එකෙහි අංකයත්, STATUS සඳහා HIGH හෝ LOW ලෙසත් ලබා දීමෙන් එම Pin එකෙහි විභවය අඩු හෝ වැඩි කිරීම කළ හැකිය.

මෙහිදීද පෙර සිටම Pin එක භාවිත කරනුයේ ආදානයක් හෝ ප්‍රතිදානයක් සඳහාද යන්න ලබා දිය යුතුය. ඔබ LEDයක් දැල්වීමට හෝ නිවීමට යම් Pin එකක් භාවිත කරයි නම්, ආරම්භයේදී එය OUTPUT Pin එකක් බව හඳුන්වා දිය යුතුය.

අදාළ Pin එකක විභවය අඩුද වැඩිද යන්න කියවීමට digitalWrite(PIN_ID) කේතය භාවිතා වේ. එය බෙහෝ විට IF Statment එකක් ආධාරයෙන් භාවිත වේ.

REQUIREMENTS
1 x ARDUINO BOARD
SETUP
Connect UNO Board to your PC 
void setup() {
  pinMode(8, OUTPUT);
  pinMode(10, OUTPUT);
}

void loop() {
  digitalWrite(8, HIGH);
  digitalWrite(10, LOW);

  int pin_st = -1;
  if (digitalRead(8) == HIGH) {
    pin_st = 1;
  }
  if (digitalRead(8) == LOW) {
    pin_st = 1;
  }
}
