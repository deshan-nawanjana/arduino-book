කේතනයේ පළමු පියවර

By Deshan Nawanjana

මෙහි දැක්වෙනුයේ Ardiuno ක්‍රමලේඛයක් සඳහා තිබිය යුතු අන්වාර්යම කේත භණ්ඩයයි. මෙහි setup සහ loop ලෙස functions කොටස් දෙකක් පවතී. ඉන් setup කොටස් යනු පුවරුව බලගැන්වූ වහාම ක්‍රියාත්මක වන function එකයි. ඉන් පසුව loop නම් function එක නොකඩවා පුනරාවර්තීව ක්‍රියාත්මක වේ. මේ නිසා පුවරුව ක්‍රියාත්මක වූ වහාම එය සකස් විය යුතු ආකාරය setup කොටස් තුලට ඇතුල් කළ යුතුය. පුවරුවේ එන Rests Button එක තද කළ සැනින් සිදු වනුයේද මෙම ක්‍රමලේඛය නැවත setup යන function එකෙන් ආරම්භ කර loop යන function එක වෙත පිවිසීමයි.
