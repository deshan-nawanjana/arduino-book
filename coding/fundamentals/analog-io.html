ANALOG I/O (ප්‍රතිසම ආදාන/ප්‍රතිදාන)

By Deshan Nawanjana

ප්‍රතිසම (Digital) ආදානයක් හෝ ප්‍රතිදානයක් යනු විභවය වරින් වර වෙනස් වන ප්‍රස්ථාරික අගයකි. එය විභවය වැඩිම අගයේ සිට අඩුම අගයට විචලනය කළ හැකිය නිදසුනක් වශයෙන් LDR සංරචකය සළකමු. LDR මගට වැටෙන ආලෝක ප්‍රමාණයේ විඩලනය අනුව එහි ප්‍රතිරෝධය වෙනස් වේ. මේ නිසා එය හරහා විභවය විචලනය වේ. එම විභව විචලනය ප්‍රතිසම අගයක් ලෙස UNO පුවරුවේ Pin එකක් මගින් කියවීමේ හැකියාව ඇත. ප්‍රතිසම අගයක් කියවීමේදී analogRead(PIN_ID) කේතය භාවිතා වන අතර එහි අගය ලෙස ලැබෙනුයේ 0 සහ 1023 යන පූර්ණ සංඛ්‍යා පරාසයේ අගයකි. එම ලැබෙන අගය අදාළ Pin එකෙහි විභිව මට්ටමට අනුරූප වේ. (ලැබෙන අගය 0 නම් Pin එකෙහි විභවය අවම වේ. ලැබෙන අගය 1023 නම් Pin එකට ලැබිය හැකි උපරිම විභිවය පවතී.) analogWrite(PIN_ID,VALUE) මගින් යම් Pin එකක විභව අගයක් ලබා දිය හැකිය. මෙහි අප VALUE සඳහා ලබා දිය හැකි අගය පරාසය 0 සිට 255 දක්වා වේ. එම අගයට අනුරූප විභව අගයක් අදාළ Pin එකට ලැබේ.

මෙහිදී යෙදාගන්නේ පුවරුවේ ප්‍රතිසම කියවීම් සඳහා යෙදාගන්නා Pins වේ. (3, 5, 6, 9, 10, 11)

REQUIREMENTS
1 x ARDUINO BOARD
SETUP
Connect UNO Board to your PC 
void setup() {
  pinMode(9, OUTPUT);
  pinMode(10, OUTPUT);
}

void loop() {
  int pin_value;
  pin_value = analogRead(9);

  analogWrite(10, 84);
}
