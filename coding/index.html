Arduino කේතකරණය මුල සිට අගට

By Deshan Nawanjana

Arduino IDE මෘදුකාංගය භාවිතයෙන් පියවරෙන් පියවර ක්‍රමලේඛණය කරන ආකාරය නිදසුන් සහිතව මෙම කොටසේ ඇතුලත් වේ. මෙහිදී සෑම ඒකකයක් සමග නිදසුනක් දැක ගත හැකිය. එක් එක් ඒකකයේ පුවරුවට සම්බන්ධ කිරීමට අවශ්‍ය අමතර උපාංග ලැයිස්තුවක් සහ ඒවා සම්බන්ධ කළ යුතු ආකාරය ඇතුලත් වේ. එය සෑම විටම හොඳින් කියවන්න. සෑම විටම නිවැරදිව උපාංග පුවරුවට සම්බන්ධ කරන්න.

ආරම්භක ඒකක:

  • කේතනයේ පළමු පියවර
  • PIN MODE හඳුන්වාදීම
  • DIGITAL I/O (අංකිත ආදාන/ප්‍රතිදාන)
  • ANALOG I/O (ප්‍රතිසම ආදාන/ප්‍රතිදාන)
  • Serial Monitor සමග දත්ත හුවමාරුව
  • Serial Plotter මගින් දත්ත නිරූපණය

සරළ පරිපථ:

  • LEDයක් BLINK කිරීම
  • LDR මගින් අගයන් කියවීම
  • Ohm Meter (ඕම් මීටරය)
  • Capacitance Meter (ධාරිත්‍රක ආරෝපණය)
  • LEDයක් වේගය වෙනස් කරමින් Blink කිරීම
  • Potentiometer එකෙහි විභව බෙදුම කියවීම

සංවේදක සහ මොඩියුල:

  • INFRARED SENSOR (IR) භාවිතය
  • ULTRASONIC SENSOR (USS) භාවිතය
  • LIQUID CRYSTAL DISPLAY (LCD) භාවිතය
  • MICROPHONE SENSOR භාවිතය
  • BUZZER MODULE භාවිතය
  • ACCELEROMETER MODULE භාවිතය
  • PRESSURE SENSOR භාවිතය
  • FLAME DETECTION SENSOR භාවිතය
  • SMOKE DETECTION SENSOR භාවිතය
  • SERVO MOTOR භාවිතය
  • PASSIVE INFRARED SENSOR (PIR) භාවිතය
  • JOYSTICK MODULE භාවිතය
  • HALL EFFECT SENSOR භාවිතය
  • LASER MODULE භාවිතය
  • INFRARED RECEIVER භාවිතය
