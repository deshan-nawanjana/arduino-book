PASSIVE INFRARED SENSOR (PIR) භාවිතය

By Deshan Nawanjana

PIR සංවේදකය මගින් ඒ අසළ සිදු වන තාප චලනයක් හඳුනාගත හැකිය. බෙහෝ විට මෙවැනි සංවේදක ආරක්ෂක පද්ධති සඳහා උචිත වන්නේ මෙමගින් සත්ත්ව චලනයන් අනාවර්ණය කළ හැකි නිසාවෙනි. මෙම සංවේදකයේ Pins 3 පවතී. ඉන් දෙකක් මෙම සංවේදකයටම විදුළිය සැපයීම සඳහා වන අතර අනෙක් Pin එක එහි ප්‍රතිධානය කියවීම සඳහා වේ. චලනයක් නිරීක්ෂණය කළ සැනින් ඉහළ ප්‍රතිදානයක් ලබා දෙන අතර යම් ප්‍රමාද කාලයකින් පසුව නැවත එය යථා තත්ත්වයට පත් වේ. මෙම ප්‍රදාඹ කාලය සංවේදකයේ ඇති Tx Pot එක මගින් සහ මෙහි සංවේදීතාවය Sx Pot එක මගින් වෙනස් කළ හැකිය.

REQUIREMENTS
1 x ARDUINO BOARD
1 x PASSIVE INFRARED SENSOR
SETUP
PIR(VCC) = 5V
PIR(OUT) = PIN(2)
PIR(GND) = GND 
const int PIRPin = 2;

void setup() {
  Serial.begin(9600);
  pinMode(PIRPin, INPUT);
}

void loop() {
  int PIRRead = digitalRead(PIRPin);
  if (PIRRead == HIGH) {
    Serial.println("Motion Detected !");
  } else {
    Serial.println("No Detection.");
  }
}
