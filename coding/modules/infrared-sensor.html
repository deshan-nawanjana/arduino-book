INFRARED SENSOR (IR) භාවිතය

By Deshan Nawanjana

Infrared Sensor මගින් ඉදිරියෙන් ඇති අඳුරු සහ ආලෝකමත් පෘෂ්ඨ හඳුනාගත හැකිය. මෙහි ඇති කලු පැහැති IR දියෝඩයෙන් නිකුත් වන කිරණය ඉදිරියේ ඇති පෘෂ්ඨයක වැදී නැවත පරාවර්ථනය වී Photodiode (අවර්ණ දියෝඩය) වෙත පැමිණේ. ඉදිරියෙන් ඇත්තේ අඳුරු පෘෂ්ඨයක් නම් ගමන් කළ කිරණ පරාවර්ථනය නොව්. එය පරාවර්ථනය වන්නේ ආලෝකමත්ල සුදට හුරු පෘෂ්ඨ මගින් පමණි. එම නිසා IR Sensor මගින් කලු සහ සුදු වර්ණ පහසුවෙන් හඳුනාගත හැකිය. කිරණ නැවත පැමිණේ නම් ප්‍රතිදානය LOW සහ නොමැති නම් HIGH ලෙසද ප්‍රතිදාන Pin එක වෙත ලැබේ.

REQUIREMENTS
1 x ARDUINO BOARD
1 x IR SENSOR
SETUP
IRS(VCC) = 5V
IRS(GND) = GND
IRS(OUT) = PIN(8) 
void setup() {
  Serial.begin(9600);
}

void loop() {
  int x = digitalRead(8);
  if (x == LOW) {
    Serial.println("IR Received (LIGHT SURFACE)");
  } else {
    Serial.println("IR NOT Received (DARK SURFACE)");
  }
}
