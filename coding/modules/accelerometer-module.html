ACCELEROMETER MODULE භාවිතය

By Deshan Nawanjana

Accelerometer මොඩියුලය, එම මොඩියුලයේ තිරස්ව හා සිරස්ව භ්‍රමණය වී ඇති ප්‍රමාණය ගණනය කළ හැකි අපූර්ව මොඩියුලයකි. ජංගම දුරකථනවල එන සමහර පරිගණක ක්‍රීඩා සඳහා (බොහෝ විට ධාවන ක්‍රීඩා) දුරකථනය දෙපසට හැරවීම සිදුකළ යුතුය. මෙම මොඩියුලයද එවැනි ක්‍රීඩා සඳහා ආදාන උපාංගයක් ලෙස භාවිත කළ හැකිය. එමෙන්ම කම්පනය වීම් පවා මෙම මොඩියුලයෙන් නිරීක්ෂණය කළ හැකිය. මෙහිදී මා භාවිතා කර ඇත්තේ ADXL345 යන මොඩියුලය බැවින් එයට අදාළව Library File එකක් යොදාගැන තිබේ. මෙ නිසා ප්‍රථමයෙන් එම ගොනුව ඔබේ Arduino IDE හි මෘදුකාංග ගොනුවලට ඇතුලත් කළ යුතුය.

Download ADXL345 Library
REQUIREMENTS
1 x ARDUINO BOARD
1 x ACCELEROMETER MODULE
SETUP
ACC(VCC) = ACC(CS) = 3V3
ACC(SDA) = PIN(A4)
ACC(SCL) = PIN(A5) 
#include <Wire.h>
#include <ADXL345.h>

ADXL345 adxl;
int x, y, z;

void setup() {
  Serial.begin(9600);
  adxl.powerOn();
}

void loop() {
  adxl.readAccel( & x, & y, & z);
  x = x / 10;
  y = y / 10;
  z = z / 10;
  Serial.print(x);
  Serial.print(" : ");
  Serial.print(y);
  Serial.print(" : ");
  Serial.println(z);
}
