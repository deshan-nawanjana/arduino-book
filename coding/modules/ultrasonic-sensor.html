ULTRASONIC SENSOR (USS) භාවිතය

By Deshan Nawanjana

මෙම Sensor එක මගින් එයට ඉදිරියන් ඇති පෘෂ්ඨයට දුර මැනීමට යොදා ගත හැකිය. මෙයින් නිතුක් කරන ශබ්ද තරංගයක පරාවර්ථනය මේ සඳහා යොදා ගැනේ. මෙහි එන Trigger Pin එක මගින් ශබ්ද තරංග ස්ඵන්ධයක් පිට කෙරේ. මෙහි කාල පරාසය මයික්‍රො තත්පර 10ක් ලෙස මෙම නිදසුනේ ලබා දී ඇත. එම කාල පරාසයටද ගණනය කිරීමට යොදා ගැනේ. Echo Pin එක මගින් එම නැවත පැමිණෙන ස්ඵන්දයේ කාල පරාසය ගෙන "වේගය = දුර / කාලය" යන සමීකරණය ආධාරයෙන් අදාළ දුර ගණනය කෙරේ. (වාතයේ ධ්වනි වේගය: තත්. මීටර 340 සහ කාලය තරංගය යාමට සහ ඒමට නිසා 2න් බෙදිය යුතුය.) මෙම නිදසුනේ ගණනය කිරීම් මගින් දුර ලබා දෙනුයේ සෙන්ටිමීටර වලිනි. එමෙන්ම මෙහි ආරම්භයේදී මිලිතත්පර දෙකක් Pin එක වසා තැබීමෙන් දෝෂ අවම කෙර්.

REQUIREMENTS
1 x ARDUINO BOARD
1 x ULTRASONIC SENSOR
SETUP
USS(VCC) = 5V
USS(TRG) = PIN(10)
USS(ECH) = PIN(9)
USS(GND) = GND 
int trigPin = 10;
int echoPin = 9;

void setup() {
  Serial.begin(9600);
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
}

void loop() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);

  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  long duration = pulseIn(echoPin, HIGH) / 2;

  int distance = (duration * 340 * 100) / 1000000;
  Serial.println(distance);
}
