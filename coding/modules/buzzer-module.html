BUZZER MODULE භාවිතය

By Deshan Nawanjana

ශබ්දය නිර්මාණය කළ හැකි මොඩියුලටක් වන BUZZER MODULE මගින් අප එයට ඇතුලත් කරන සංඥාවට අනුරූපව වෙනස් ආකාරයේ ශබ්දයන් නිරීක්ෂණය කළ හැකිය. මෙම මොඩියුලයේ එන I/O Pin එක මගින් අප ඉදිරිපත් කරන සංඥාව මොඩියුලයට ලබා දේ. මෙහිදී එම Pin එක මත විභවය අඩු හා වැඩි කිරීම සිදු කරයි. එම අඩු හැ වැඩි කිරීමෙහි ප්‍රමාද කාලය (Delay Time) මගින් අපගේ සංඥාව වෙනස් කළ හැකිය. මේ නිසා මෙහි එන delayTime විචල්‍යයෙහි අගය වෙනස් කිරීමෙන් මොඩියුලය පිට කරන ශබ්දයද වෙනස් කළ හැකිය. එමෙන්ම මෙහි I/O Pin එක හරහා ඔබට අවශ්‍ය ප්‍රතිරෝධයක් යෙදීමෙන් හෝ Potentiometerක් යෙදීමෙන් එහි හඬ පාලනය කිීරීමටද හැකියාව ඇත.

REQUIREMENTS
1 x ARDUINO BOARD
1 x BUZZER MODULE
SETUP
BUZ(VCC) = 5V
BUZ(GND) = GND
BUZ(I/O) = PIN(3) 
int buzzerPin = 3;
int delayTime = 2;

void setup() {
  pinMode(buzzerPin, OUTPUT);
}

void loop() {
  digitalWrite(buzzerPin, HIGH);
  delay(delayTime);
  digitalWrite(buzzerPin, LOW);
  delay(delayTime);
}
