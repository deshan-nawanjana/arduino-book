MICROPHONE SENSOR භාවිතය

By Deshan Nawanjana

ශබ්දයට සංවේදී Microphone Sensor මගින් එම සංවේදකයට ශ්‍රවණය වන ශබ්ද තරංගයට අනුරූපව ප්‍රතිසම සංඥාවක් නිකුත් කිරීමේ හැකියාව ඇත. එම ප්‍රතිසම කියවීම Arduino හි Analog Pin එකක් ආධාරයෙන් සරළව කියවා ගත හැකිය. එම කියවන අගය ශ්‍රවණය වන උබ්දයට අනුරූපව 0 සහ 1023 අතර අගයක් වේ.

REQUIREMENTS
1 x ARDUINO BOARD
1 x MICROPHONE SENSOR
SETUP
MIC(VCC) = 5V
MIC(GND) = GND
MIC(OUT) = PIN(A0) 
void setup() {
  Serial.begin(9600);
}

void loop() {
  int x = analogRead(A0);
  Serial.println(x);
}
