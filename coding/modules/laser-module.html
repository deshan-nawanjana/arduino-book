LASER MODULE භාවිතය

By Deshan Nawanjana

රතු ලේසර් ආලෝකයක් නිකුත් කිරීමේ හැකියව Laser මොඩියුලය සතු වේ. මෙහි Pin තුනක් පවතින අතර ඉන් දෙකක් මොඩියුලයට අවශ්‍ය විදුලිය සැපයීම සඳහා වන අතර අනෙක මගින් දැල්වීම සහ නිවීම සිදු කරයි. මෙම Pin එක Signal Pin නම් වන අතර එහි ආදානය ඉහළ කළ විට ලේසරයට ක්‍රියාත්මක වේ. මෙම නිදසුනේ Laser මොඩියුලය තත්පරයට වරක් නිවි නිවී දැල්වෙන ආකාරයෙන් කේතනය කර ඇත.

REQUIREMENTS
1 x ARDUINO BOARD
1 x LASER MODULE
SETUP
LSR(VCC) = 5V
LSR(GND) = GND
LSR(SIG) = PIN(8) 
const int lsrPin = 8;

void setup() {
  pinMode(lsrPin, OUTPUT);
}

void loop() {
  digitalWrite(lsrPin, HIGH);
  delay(1000);
  digitalWrite(lsrPin, LOW);
  delay(1000);
}
