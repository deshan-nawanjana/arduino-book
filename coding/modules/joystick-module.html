JOYSTICK MODULE භාවිතය

By Deshan Nawanjana

බොහෝ විට පරිගණක ක්‍රීඩා කටයුතු සඳහා භාවිතා වන Joystick මොඩියුලට මගින් ප්‍රධාන දිශා 4කට (ඉහළ, පහළ, වම සහ දකුණ) ලෙස කියවීම් ලබා ගැනේ. එමෙන්ම එහි තද කළ හැකි බටනයක්ද වේ. මේ ආකාරයට ප්‍රතිදාන ලබා ගැනීම සඳහා මෙම මොඩියුලයේ Pin 3 ක් සහ එයට අවශ්‍ය විදුලිය සැපයීමට Pin 2 ක්ද පවතී. දිශා සඳහා ප්‍රතිදාන ලබා දීමට Pin දෙකක් වේ. මෙම ප්‍රතිදාන ප්‍රතිසම ආකාරයේ ඒවා වන අතර ඉහළ සහ පහළ සඳහා එක් ප්‍රතිසම ප්‍රතිදානයක් (0 සහ 1023 අතර) ලබා දේ. මොඩියුලට ඉහළට හෝ පහළට චලනය කර නොමැති නම් එම ප්‍රතිදාන අගය මධ්‍ය අගයක් (500 ට ආසන්න) වේ. එය ඉහළට ගෙන ගිය විට එහි අගය අවම (0) වේ. පහළට ගෙන ගිය විට උපරිම (1023) වේ. වම සහ දකුණ සඳහාද මේ ආකාරයෙන් කියවීම් ලබා ගත හැකිය. බටනය සඳහා එක් Pin එකක් වෙන් වී ඇති අතර බටනය තද කළ විට එහි ප්‍රතිදාන අගය උපරිම වේ.

REQUIREMENTS
1 x ARDUINO BOARD
1 x JOYSTICK MODULE
SETUP
JST(GND) = GND
JST(VCC) = 5V
JST(VRX) = PIN(A0)
JST(VRY) = PIN(A1)
JST(SW) = PIN(2) 
const int sw = 2;

void setup() {
  Serial.begin(9600);
  pinMode(sw, INPUT_PULLUP);
}

void loop() {
  int x = analogRead(A0);
  int y = analogRead(A1);
  int s = digitalRead(sw);

  Serial.print(x);
  Serial.print("	");
  Serial.print(y);
  Serial.print("	");
  Serial.println(s);
}
