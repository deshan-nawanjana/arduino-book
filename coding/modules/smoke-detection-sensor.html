SMOKE DETECTION SENSOR භාවිතය

By Deshan Nawanjana

අවට දුමක් පවතින අවස්තාවක එය හඳුනාගැනීම සඳහා මෙම Gas Detection Sensor එක භාවිතා කළ හැකිය. මෙම සංවේදකයේ අග්‍ර 4ක් පවතින අතර ඉන් දෙකක් සංවේදකයට අවශ්‍ය විදුලිය සැපයීම සඳහා වන අතර අනෙක් අ‍ග්‍ර දෙක මගින් සංකිත සහ ප්‍රතිසම ආකාරයේ ප්‍රතිදානයන් ලබා ගත හැකිය. අවට පරිසරයේ දුමක් පවතින ප්‍රතිශතය අනුව මෙහි ලැබෙන ප්‍රදිසම ප්‍රතිදානය 0 සිට 1023 අතර විචලනය වන අගයක් පෙන්වයි. නිදසුනක් ලෙස දල්වන ලද හඳුන් කූරක් මෙම සංවේදකය අසළට ගෙන ආ විට එම ප්‍රතිදාන අගය ඉහළ යයි. මේ ආකාරයෙන් ඉතා පහසුවෙන් දුමක් හඳනා ගැනීම සඳහා Gas Detection Sensor එක භාවිතා කළ හැකිය.

REQUIREMENTS
1 x ARDUINO BOARD
1 x SMOKE DETECTION SENSOR
SETUP
SDS(VCC) = 5V
SDS(GND) = GND
SDS(A0) = PIN(A0) 
void setup() {
  Serial.begin(9600);
}

void loop() {
  int x = analogRead(A0);
  Serial.println(x);
}
