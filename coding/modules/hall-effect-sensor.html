HALL EFFECT SENSOR භාවිතය

By Deshan Nawanjana

අවට ඇති චුම්භක ක්ෂේත්‍රයක් හඳුනාගැනීම සඳහා මෙම සංවේදකය භාවිත කළ හැකිය. මෙමගින් ප්‍රතිම ආකාරයෙන් හෝ අංකිත ආකායෙන් ප්‍රතිදානය කියවීමට හැකියව පවතී. මේ සඳහා AO (Analog Out) සහ DO (Digital Out) ලෙස Pin 2ක් පවතී. සංවේදකය අසලට චුම්භකයක් ගනෙ ආ විට ප්‍රතිදානය අඩු වන අතර ප්‍රබල චුම්භක ක්ෂේත්‍රයක් සංවේදකය අවට නොමැති විට ප්‍රතිදානය උපරිම වේ. චුම්භක ක්ෂේත්‍රෙය් ප්‍රභලත්වය මත එය හඳනාගැනීමට අවම වශයෙන් සංවේදකයේ සිට කොපමණ දුරකින් පැවතිය යුතුද යන්න තීරණය වේ.

REQUIREMENTS
1 x ARDUINO BOARD
1 x HALL EFFECT SENSOR
SETUP
HES(VCC) = 5V
HES(GND) = GND
HES(D0) = PIN(8) 
const int HEPin = 8;

void setup() {
  Serial.begin(9600);
  pinMode(HEPin, INPUT);
}

void loop() {
  if (digitalRead(HEPin) == HIGH) {
    Serial.println("No detection.");
  } else {
    Serial.println("Magnetic Field Detected!");
  }
}
