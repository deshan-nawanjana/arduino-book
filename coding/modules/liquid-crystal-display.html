LIQUID CRYSTAL DISPLAY (LCD) භාවිතය

By Deshan Nawanjana

Liquid Crystal Display මගින් අපිට අවශ්‍ය වැකියක් හෝ පණිවුඩයක් දර්ශණය කරගැනීමේ හැකියාව ඇත. මෙහිදී භාවිතා කර ඇත්තේ 16x2 ප්‍රමාණයේ අක්ෂර ප්‍රමාණයක් නිරූපණය කළ හැකි තිරයකි. LiquidCrystal.h යන Library ගොනුව මේ සඳහා අවශ්‍ය වේ. මෙහි එන setCursor(X,Y) මගින් අදාල වැකිය දර්ශණ කළ යුතු පිහිටීම ලබා දිය හැකිය.

REQUIREMENTS
1 x ARDUINO BOARD
1 x LIQUID CRYSTAL DISPLAY
1 x 10K Ω POTENTIOMETER
SETUP
LCD(VSS) = GND
LCD(VDD) = 5V
LCD(V0) = POT(WP)
LCD(RS) = PIN(7)
LCD(RW) = GND
LCD(D4) = PIN(9)
LCD(D5) = PIN(10)
LCD(D6) = PIN(11)
LCD(D7) = PIN(12)
LCD(A) = 5V
LCD(K) = GND
LCD(E) = PIN(8)
POT(T1) = 5V
POT(T2) = GND 
#include <LiquidCrystal.h>

LiquidCrystal lcd(7, 8, 9, 10, 11, 12);

void setup() {
  lcd.begin(16, 2);
  lcd.setCursor(0, 0);
  lcd.print("First Line Text");
}

int x = 0;

void loop() {
  x += 1;
  lcd.setCursor(0, 1);
  lcd.print(x);
  delay(1000);
}
