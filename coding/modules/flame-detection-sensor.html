FLAME DETECTION SENSOR භාවිතය

By Deshan Nawanjana

නුදුරින් ඇති ගින්නක් හඳුනාගැනීම සඳහා මෙම Flame Detection Sensor එක භාවිතා කරයි. මෙම සංවේදකයේ අග්‍ර තුනක් ඇති අතර ඉන් දෙකක් සංවේදකය බලය සැපයීම සඳහා වන අතර අනෙක් අග්‍රය මගින් ප්‍රතිදාන අංකිත අගය කියවීම සිදු කරයි. ගිණි අනතුරක් හඳුනාගත නොහැකි නම් එම ප්‍රතිදාන අගය උපරිම වන අතර ගිණි දැල්ලක් හඳුනාගත් වහිම එහි ප්‍රතිදානය අවම අගයකට පත් වේ. මේ නිසා ඉතා පහසුවෙන් Flame Detection Sensor මගින් ගිණි අනතුරක් හඳුනාගැනීමේ හැකියාව පවතී.

REQUIREMENTS
1 x ARDUINO BOARD
1 x FLAME DETECTION SENSOR
SETUP
FDS(VCC) = 5V
FDS(GND) = GND
FDS(D0) = PIN(4) 
void setup() {
  Serial.begin(9600);
  pinMode(4, INPUT);
}

void loop() {
  if (digitalRead(4) == LOW) {
    Serial.println("FIRE DETECTED!!!");
  } else {
    Serial.println("No fire detected.");
  }
}
