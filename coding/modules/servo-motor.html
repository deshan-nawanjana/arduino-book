SERVO MOTOR භාවිතය

By Deshan Nawanjana

Arduino සඳහා යොදා ගන්නා විධ ආකාරයේ මෝටර පවතින අතර ඉන් සුවිශේසී මෝටර වර්ගයක් ලෙස Servo Motor හඳුන්වා දිය හැකිය. මෙම මෝටර් වර්ගයේ විශේෂත්වය නම්, මෝටර ඇක්සලය අප ලබා දෙන නිශ්චිත කෝණයකට භ්‍රමණය කළ හැකි වීමයි. මේ සඳහා Servo.h නම් වූ Library File ඒක භාවිතා කළ යුතු අතර එය Arduino IDE මෘදුකාංගය සමගම පරිගණකයේ ස්ථාපිත වේ. මෙහිදී servo_name.attach(PIN_ID) යන කේතයෙන් මෝටරය වෙත සංඥා සපයන Pin එක ලබා දේ. මෝටරයේ අනෙක් Pin දෙක මෝටරයට අවශ්‍ය විදුලිය සැපයීමට භාවිතා වේ. මෝටරය ක්‍රියාත්මක කිරීමේ දී servo_name.write(ANGLE) යන කේතයෙන් මෝටරය කැරකැවිය යුතු කෝණය ලබා දිය හැකිය. මෙම නිදසුනේ ඇත්තේ තත්පරයෙන් 1/10 කට වරක් එහි කෝණය අංශක 10 බැගින් වෙනස් වන අයුරින් ලියූ වැඩසටහනකි.

REQUIREMENTS
1 x ARDUINO BOARD
1 x SERVO MOTOR
SETUP
SVR(VCC) = 5V
SVR(GND) = GND
SVR(IN) = PIN(10) 
#include <Servo.h>

Servo svr;

void setup() {
  Serial.begin(9600);
  svr.attach(10);
}

void loop() {
  for (int i = 0; i < 181; i += 10) {
    svr.write(i);
    delay(100);
  }
}
