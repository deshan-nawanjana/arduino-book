INFRARED RECEIVER භාවිතය

By Deshan Nawanjana

බොහෝමයක් විදුලියෙන් ක්‍රියාත්මක වන උපකරණ සඳහා (රූපවාහිනී, වායු සමීකරණ වැනි) ලැබෙන දුරස්ථ පාලකයේ ඇති සංඥා නිකුත් කරන තාක්ෂණය වන්නේ Infrared කිරණයි. මෙම කිරණය මගින් නිකුත් කරන සංඥාව හඳුනා ගැනීමට Infrared Receiver මොඩියුලයට හැකියාව ඇත. එම අගය ෂඩ් දශමට ආකාරයෙන් කියවීම සාමාන්‍යෙයන් සිදු කරයි. ඔබ ලඟ ඇති ඕනෑම දුරස්ථ පාලකයක එක් එක් බටන සඳහා නිකුත් කරණ සංඥා මෙමගින් හඳුනා ගත හැකි අතර, Infrared Transmitter මගින් එම ආකාරයට සංඥාව නිකුත් කිරීමේ හැකියව ඇත. මේ නිසා දුරස්ථ පාලකය නොමැතිව වුවද අදාළ උපකරණය දුරස්ථව පාලනය කළ හැක. මේ සඳහා භාවිතා කළ යුතු Library ගොනුව පහතින් ලබා ගන්න.

Download IRremote Library
REQUIREMENTS
1 x ARDUINO BOARD
1 x INFRARED RECEIVER
SETUP
IRR(VCC) = 5V
IRR(GND) = GND
IRR(SIG) = PIN(7) 
#include <IRremote.h>

const int irPin = 7;
IRrecv irrecv(irPin);
decode_results results;

void setup() {
  Serial.begin(9600);
  irrecv.enableIRIn();
  irrecv.blink13(true);
}

void loop() {
  if (irrecv.decode( & results)) {
    Serial.println(results.value, HEX);
    irrecv.resume();
  }
}
