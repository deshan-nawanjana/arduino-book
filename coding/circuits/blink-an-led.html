LEDයක් BLINK කිරීම

By Deshan Nawanjana

ඕනෑම Digital Pin එකක් සහ Ground Pin එකක් අතරට LEDයක් සම්බන්ධ කරන්න. කේතයේ පරිදි අදාළ Pin එකෙහි විභවය වරින් වර වැඩි කිරීම සහ අඩු කිරීම මගින් LEDය Blink කළ හැකිය. එමෙන්ම විභවය වැඩි හෝ අඩු කිරීමෙන් පසු ලබා දෙන Delay Time එක මගින් එහි Blink රටාව සහ වේගය වෙනස් කළ හැක. ආරම්භයේදී, යෙදාගන්නා Pin එක Output Pin එකක් බවද හඳුන්වා දිය යුතුය.

REQUIREMENTS
1 x ARDUINO BOARD
1 x LED
SETUP
LED(T+) = PIN(13)
LED(T-) = GND 
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(100);
  digitalWrite(13, LOW);
  delay(100);
}
