Potentiometer එකෙහි විභව බෙදුම කියවීම

By Deshan Nawanjana

Potintiometer තුළ ඇති විභව බෙදීම එහි පිහිටීම අනුව සිදු වී ඇති අයුරු කියවීම මෙම ක්‍රමලේකණය තුළින් සිදු වේ. මේ සඳහා මා විසින් 10K Ω Potintiometerක් යොදා ඇති අතර ඔබට අවශ්‍ය අගයක Potintiometerක් යෙදා ඊට අදාළව කේතයේ ඇති potResistance අගය වෙනස් කිරීමෙන් මෙම පරිපථය ක්‍රියාත්මක කළ හැකිය. මේ සඳහා යොදාගෙන ඇති මූලික සංකල්පය විභව බෙදුම් මූලධර්මයයි (V1/R1 = V2/R2) පරිපථය ක්‍රියාත්මක වන මොහොතේ Potintiometer එකෙහි Wiper එක කැරකැවීමෙන් එම අවස්ථාවේදීම ඒ හරහා විභවය හා ප්‍රතිරෝධය බෙදී ඇති අයුරු Serial Monitor මගින් මෙහිදී බලා ගත හැකිය.

REQUIREMENTS
1 x ARDUINO BOARD
1 x 10K Ω POTENTIOMETER
SETUP
POT(T1) = 5V
POT(WP) = PIN(A0)
POT(T2) = GND 
float potResistance = 10000.0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int ax = analogRead(A0);
  float v1 = (ax * 5) / 1024.0;
  float v2 = ((1023 - ax) * 5) / 1024.0;

  float r1 = (v1 * potResistance) / (v1 + v2);
  float r2 = (v2 * potResistance) / (v1 + v2);

  Serial.print("V1 = ");
  Serial.print(v1);
  Serial.print(" , ");
  Serial.print("V2 = ");
  Serial.print(v2);
  Serial.print("  |  ");
  Serial.print("R1 = ");
  Serial.print(r1);
  Serial.print(" , ");
  Serial.print("R2 = ");
  Serial.println(r2);
}
