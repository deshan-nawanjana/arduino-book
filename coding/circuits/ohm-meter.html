Ohm Meter (ඕම් මීටරය)

By Deshan Nawanjana

ප්‍රතිසම සංඥා කියවන Pin එකක් සමග දන්නා ප්‍රතිරෝධයක් සහ නොදන්නා ප්‍රතිරෝධයක් සම්බන්ධ කිරීමෙන් නෙදන්නා ප්‍රතිරෝධයේ අගය ගණනය කළ හැකිය. මේ සඳහා විභව බෙදුම් මූලධර්මය (V1/V2=R1/R2) භාවිතා කර තිබේ. දන්නා ප්‍රතිරෝධය A0 Pin එක සහ GND අතරට සම්බන්ධ බැවින් A0 සඳහා ලැබෙන්නේ දන්නා ප්‍රතිරෝධය හරහා විභව අන්තරයයි. (විභව අන්තරයට අනුරූප 0 - 1023 අතර අගය) මුලු විභව සැයපුම වන 5V අගයෙන් එම අගය අඩු කළ විට නොදන්නා ප්‍රතිරෝධය හරහා විභව අන්තරය ලැබේ. මේ ආකාරයෙන් අදාළ අගයන් ගෙන සමීකරණයට යෙදීමෙන් නොදන්නා ප්‍රතිරෝධයේ අගය ගණනය කළ හැකිය. මෙහිදී ගණනය කරන අගය වඩාත් නිවැරදි වන්නේ දන්නා සහ නොදන්නා ප්‍රතිරෝධයන්ගේ අගයන් අතර පරතරය අවම වන විටදී ය.

REQUIREMENTS
1 x ARDUINO BOARD
1 x 1k Ω RESISTOR (R1)
UNKNOWN RESISTORS (R2)
R1(T1) = GND
R1(T2) = R2(T1) = PIN(A0)
R2(T2) = 5V
Connect UNO Board to your PC 
int APin = A0;
float V0 = 5.0;

float R1 = 1000.0;
float R2 = 0.0;
float V1 = 0.0;
float V2 = 0.0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  float V1 = analogRead(APin) * V0 / 1024.0;
  float V2 = V0 - V1;
  float R2 = R1 * (V2 / V1);
  Serial.println(R2);
}
