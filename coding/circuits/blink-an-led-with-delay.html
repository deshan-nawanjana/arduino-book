LEDයක් වේගය වෙනස් කරමින් Blink කිරීම

By Deshan Nawanjana

Potintiometer මගින් එහි දෙපස අග්‍ර දෙකට විභව බෙදීම විචලනය කළ හැකිය. මේ නිසා එහි මධ්‍ය අග්‍රයෙහි විභව අගය කියවීමෙන් 0 සිට 1023 දක්වා වෙනස් වන අගයක් ලබා ගත හැකිය. එම අගය මෙම කේතයේ 4න් බෙදා ඇත්තේ උපරිම මිලිතත්පර 256 ක Blink Delay එකක් ලබා ගැනීමටයි. පරිපථය පහත පරිදි සකස් කළ පසු Potintiometer හි වයිපරය කැරකැවීමෙන් LDEය Blink වන වේගය වෙනස් කර ගත හැකිය. මෙහි Resistor එකක් යොදා ඇත්තේ LEDයේ ආරක්ෂාව සඳහායි.

REQUIREMENTS
1 x ARDUINO BOARD
1 x 10K Ω POTENTIOMETER
1 x 100 Ω RESISTOR
1 x LED
SETUP
LED(T+) = RES(T1)
RES(T2) = PIN(13)
POT(T1) = 5V
POT(WP) = PIN(A0)
LED(T-) = GND
POT(T2) = GND 
int blinkSpeed = 0;

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  blinkSpeed = analogRead(A0) / 4;
  digitalWrite(13, HIGH);
  delay(blinkSpeed);
  digitalWrite(13, LOW);
  delay(blinkSpeed);
}
