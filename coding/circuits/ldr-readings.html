LDR මගින් අගයන් කියවීම

By Deshan Nawanjana

ආලෝක සංවේදී දියෝඩයක් (LDR) මගින් ඒ මත පතිත වන ආලෝකයට අනුරූපව 0 සිට 1023 දක්වා වූ සංඛ්‍යාත්මක අගයක් ලබා ගත හැකිය. එම කියවීම Analog ගණයට වැටේ. මේ නිසා දත්ත ලබා ගැනීමට Analog Pin එකක් බාවිත කළ යුතුය. මෙම නිදසුනේ පරිදි ප්‍රතිදාන අගය පරිගණකයේ Arduino Serial Monitor මත දර්ශණය වේ. මේ සඳහා ප්‍රතිරෝධයක්ද යොදාගත යුතු වේ.

REQUIREMENTS
1 x ARDUINO BOARD
1 x LDR
1 x 680 Ω RESISTOR
SETUP
LDR(T1) = 5V
LDR(T2) = RES(T2) = A0
RES(T1) = GND 
void setup() {
  Serial.begin(9600);
}

void loop() {
  int x = analogRead(A0);
  Serial.println(x);
}
