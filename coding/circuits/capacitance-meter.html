Capacitance Meter (ධාරිත්‍රක ආරෝපණය කියවීම)

By Deshan Nawanjana

ධාරිත්‍රකයකට විදුලිය සැපයූ සැනින් ඒය ආරෝපණය වීමට පටන් ගනිිිියි. ආරම්භයේදී එය ඉතා වේගයෙන් ආරෝපණය වන අතර අවසානයේ එහි ආරෝපණ ‌වේගය මන්දගාමී ‌වේ. මෙම ආරෝපණය වන වේගය ප්‍රතිශතයක් ලෙස මෙම පරිපථයෙන් ගණනය කළ හැකිය. ධාරිත්‍රකය ප්‍රතිරෝධයක් හරහා සම්බන්ධ කර ඇති නිසා ආරෝපිත වේගය සමස්ථයක් වශයෙන් අඩු වන බැවින් එහි විචලනය හොඳින් දැකගත හැකි වේ.

පහතින් දැක්වෙන ආකාරයට පරිපථය සම්බන්ධ කල විට ධාරිත්‍රකය ආරෝපණය වීම පෙන්වන අතර RES(T2) = PIN(13) සම්බන්ධය ඉවත් කර RES(T2) = CAP(T-) ලෙස සම්බන්ධ කිරීමෙන් ධාරිත්‍රකය විසර්ජනය වන ප්‍රතිශතයද බලා ගත හැකිය.

REQUIREMENTS
1 x ARDUINO BOARD
1 x 100 µF CAPACITOR
1 x 10 kΩ RESISTOR
SETUP
CAP(T+) = RES(T1) = PIN(A0)
RES(T2) = PIN(13)
CAP(T-) = GND 
void setup() {
  Serial.begin(9600);
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  float prc = (analogRead(A0) / 1024.0) * 100.0;
  Serial.println((String)(prc) + "%");
}
